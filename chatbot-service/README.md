# Chatbot Service

## Chức năng
- Chat với AI về nhu cầu build PC
- Trả lời câu hỏi kỹ thuật
- Lưu lịch sử hội thoại

## Cài đặt
```bash
npm install
npm run dev
```

## Biến môi trường
- PORT
- MONGO_URI
- OPENAI_API_KEY

## Endpoint chính
- POST /chat/message
- GET /chat/history

## Công nghệ
- Node.js, Express, MongoDB, OpenAI API
